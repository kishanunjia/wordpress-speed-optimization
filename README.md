## Local Preview

**Option 1 — XAMPP**

Place the project in your `htdocs` folder and open:

```
http://localhost/speed-optimization/
```

**Option 2 — Open directly**

Open `index.html` in your browser (some features work best via a local server).

## Deploy to GitHub Pages

1. Push this repository to GitHub
2. Go to **Settings → Pages**
3. Set **Source** to `Deploy from branch`
4. Select branch `main` and folder `/ (root)`
5. Save — your site will be live at `https://YOUR_USERNAME.github.io/wordpress-speed-optimization/`

> Update the canonical URL in `index.html` if your GitHub username differs from `kishanunjia`.
